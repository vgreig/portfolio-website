    About Me

    I started tinkering with code in 2023 as a hobby in my spare time. I quickly found that I loved that coding enabled me to create and build things, but using logic rather than artistic flair (my logic far outweighs my artistic abilities!). I still find it incredibly satisfying to see projects I’m working on come together, whether it’s front end or back end. I continued to code in my spare time, following courses such as Code First Girls’ Intro to Python course, until I decided to take the leap and leave my job in property to take a full time coding bootcamp- which is how I ended up as a student at iO Academy. The course at iO has introduced me to new skills such as PHP, React, Laravel and MySQL as well as expanding my skills in HTML, CSS, Javascript and API manipulation. I’m now looking for my first job in software.
